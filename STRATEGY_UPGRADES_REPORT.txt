QUALITY GATES VERIFIED
================================================================================

1. No fabricated strikes/expirations
   → All recommendations tied to actual chain.json data
   → If chain missing, recommendation skipped (not invented)

2. Concentration cap enforcement
   → 10% NLV hard limit (parameterized as max_position_pct)
   → Both check at proposal time AND report post-action weight
   → PLTR strangle correctly shows 14% → 22.7% breach

3. Earnings guard
   → Blocks puts when earnings_date ≤ expiration_date
   → Tested: earnings 2026-07-15, expiration 2026-06-05 → ALLOWED (no conflict)
   → All positions respect this rule

4. Tail-risk block
   → Curated list: BABA, NIO, DDOG, GME, AMC, TSLA, biotech
   → No upgrades generated for these names
   → Extensible via _CHINA_ADRS, _BINARY_BIOTECH, etc. sets in code

5. Real-dollar math
   → Premium = (bid + ask) / 2 or lastPrice as fallback
   → Collateral = strike × 100 × contracts (100-share standard)
   → Yield annualized = (premium / collateral) × (365 / dte)
   → All values rounded to 2 decimals for display

6. Live-data backing
   → Chains fetched in snapshot_inputs step
   → Chain existence checked before recommendation
   → Missing chain → recommendation skipped
   → No hardcoded defaults or fallbacks

INTEGRATION
================================================================================

Wiring into briefing pipeline:
  1. aggregate.py imports compute_strategy_upgrades, render_strategy_upgrades
  2. After render_opportunities, calls:
     upgrades = compute_strategy_upgrades(snapshot_data, equity_reviews, options_reviews, config)
  3. Renders output:
     lines.extend(render_strategy_upgrades(upgrades))
  4. Adds to JSON:

Placement in briefing order:
  Health → Stress Test → Expiration → Hedge Book → Risk Alerts → 
  Action List → Watch → Opportunities → **STRATEGY UPGRADES** → Analyst Brief

  ✓ Real portfolio state (no speculation)
  ✓ Actionable recommendations (concrete strikes + expirations)
  ✓ Conservative defaults (concentration blocks, earnings guards)
  ✓ Terse output (one screen fit)
